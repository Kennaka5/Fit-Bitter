This is a full Stack React Native application named "tracks"

as of 10/28/2019 this application was built on React-native.

This app is modeled as a fit bit.  A user can create an account, and use the app to track there movements on a map.

When recording the app will draw a line indicating where the user moved.

The User can save there "tracked" path to there account and revew there past movements.

The back end portion of this app is built in the "track-server" Repository.

npm start to start up application

use ngrok for testing with mobile device to the express server
to start up nrgok: ngrok http 30000
